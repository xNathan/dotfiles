# Dotfiles

个人使用的 dotfiles。

# 安装：

```bash
$ git clone https://github.com/xNathan/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ pipenv install
$ pipenv run ./dotdrop.sh install -p mac
# 如果是 Archlinux，运行：
$ pipenv run ./dotdrop.sh install -p arch
```

# 依赖：

Python 3.6+
